1.0 Pendahuluan

Latar Belakang

Transisi energi di Indonesia memicu lonjakan minat pada saham EBT. Namun, volatilitas tinggi seringkali mengaburkan fundamental. Investor membutuhkan metode kuantitatif untuk membedakan pertumbuhan organik vs manipulasi pasar.

Objektif Analisis

  • ▸
    Akuisisi Data Valid: Mengambil data *Time Series* harian langsung dari server IDX (bukan PDF).
  • ▸
    Deteksi Anomali: Mengidentifikasi "Data Burst" menggunakan statistik Z-Score.
  • ▸
    Analisis Pola: Menguji keberadaan siklus musiman dengan Transformasi Fourier (FFT).

2.0 Metodologi (Data Science)

Mengadaptasi *Best Practices* dari studi literatur Kaggle untuk memastikan ketajaman analisis.

1. Preprocessing

Interpolasi Linear untuk menangani data hari libur, memastikan kontinuitas *time-series* sebelum masuk ke model FFT.

2. Z-Score Anomaly

Threshold Z > 2.5. Menyaring fluktuasi normal untuk menemukan kejadian ekstrem (outlier).

Ref: Praxitelis K (Kaggle)

3. Spectral Analysis

Mengubah domain waktu ke frekuensi (Hz) untuk memisahkan *Signal* (Pola) dari *Noise* (Acak).

Ref: Ian Olmstead (Kaggle)
