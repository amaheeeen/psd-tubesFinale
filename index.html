2.0 Mathematical Framework

Analisis ini didasarkan pada dua pilar statistik yang kokoh (Referensi: Kaggle Best Practices).

A. Normalized Returns

Kami tidak menggunakan harga mentah (Pt) karena tidak stasioner. Kami menggunakan Logarithmic Returns untuk menormalkan distribusi data:

Rt = ln( Pt / Pt-1 )

*Ini memungkinkan perbandingan adil antara saham harga Rp 2.000 (SLIS) dan Rp 700 (POWR).

B. Anomaly Detection (Z-Score)

Kami mendefinisikan "Anomali" sebagai kejadian dengan probabilitas < 1%. Menggunakan asumsi distribusi normal pada rolling window 20 hari:

Z = (x - μ) / σ

Threshold: Jika |Z| > 2.5, maka sinyal bahaya (Burst) aktif.
